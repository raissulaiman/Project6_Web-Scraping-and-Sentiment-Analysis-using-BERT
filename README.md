# Project6_Web-Scraping-and-Sentiment-Analysis-using-BERT
Sentiment Analysis Using IndoBERT with Data from Tokopedia Web Scraping

🔎 **The goal** of this project is to create a sentiment analysis model using IndoBERT to predict a review's sentiment. 

🗂️ **The dataset** of the model is from https://www.sciencedirect.com/science/article/pii/S2352340922007612, and for the fine-tuned model, tha data was extracted from tokopedia web scraping.

⚙️ **The methodology:**

1️⃣ The data for the model will be extracted from the link above

2️⃣ Conducting Exploratory Data Analysis (EDA)

3️⃣ Developing the model with IndoBERT 

4️⃣ Creating a pipeline for sentiment analysis with a fine-tuned model with the data from tokopedia web scraping

💡 **Insight**
IndoBERT is an algorithm that excels at predicting the sentiment of Indonesian text. 

🚨**Recommendation**
Create a pipeline to predict review sentiment and understand customers' emotions when they review products.

🛠️ **Tools:**
Python, Matplotlib, Numpy, Pandas, WordCloud, nltk, transformers (Hugging Face), TensorFlow, torch
